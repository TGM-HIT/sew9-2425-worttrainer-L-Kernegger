# Worttrainer
## Grundlegend
Es wird ein Folder angegeben und dann alle Bilder in
diesem Folder geladen, und mit dem Namen dieser Bilder
als bezeichnung des Bildes versehen

## Modi
Es gibt zwei Modi zwischen denen man mit `ctr+alt+s` switchen kann.
Der erste ist eine freie Text eingabe, gut für wenn der Name leicht ersichtlich ist.
Zweiter Modi gibt dem Nutzer 4 Auswahlmöglichkeiten, was es einfacher
macht schwierigere oder unklare Wörter richtig zu kriegen.
